=== README - LiftLog ===

prerequisites
- Install Visual Studio
- Install .NET Core binaries: https://www.microsoft.com/net/download/core
- Install npmJS: https://www.npmjs.com/get-npm?utm_source=house&utm_medium=homepage&utm_campaign=free%20orgs&utm_term=Install%20npm

== SETUP ==
- Run the SETUP.bat in main directory


== START ==
- Open LiftLog.sln in Visual studio
- Set LiftLog.Web as startup project
- Launch program
